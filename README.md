## 项目介绍

* 开发环境：```Android Studio 4.2.1```

* 这是一款```翻译APP```，可以快捷无广告的进行文本翻译，做到打开即用的效果（只有初次打开会显示封面）。
* 用户无需点击翻译按钮，可以做到输入即自动翻译。
* 目前只完成了一小部分，后续会逐步完成收藏和我的页面。
