## 项目介绍

* 开发环境：```Android SdkVersion 30```，```SpringBoot2.5.3```
* 这是一款```翻译APP```，可以快捷的进行文本翻译，做到打开即用的效果。
* 使用``Spring Boot``进行前后端分离，利用```OkHttp```接入，集成了登录系统。
* 用户无需点击翻译按钮，可以做到输入即翻译。
